# Build a Student Database

This workshop is part of the freeCodeCamp Relational Datatabase. 

## Description
A PostgreSQL database project that stores student and course information. Data is imported from CSV files using a Bash script and inserted into a relational database.

## Files
- `students.sql` - Database schema and data dump
- `insert_data.sh` - Script used to import data into the database
- `student_info.sh` - Script used to query and display student information 
- `courses.csv` - Course data source
- `students.csv`- Student data course
