# Build a Student Database

This workshop is part of the freeCodeCamp Relational Datatabase. In this 140-lesson workshop, I created a Bash script that uses SQL to enter information about computer science students into PostgreSQL.

## Files
- `students.sql`
- `insert_data.sh`
- `courses.csv`
- `students.csv`

## Technologies
- Bash
- PostgreSQL
- SQL
